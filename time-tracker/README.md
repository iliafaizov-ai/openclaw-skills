# Time Tracker Skill

Автоматический time tracking для OpenClaw с регулярными опросами, группировкой активностей и недельной статистикой.

## Быстрый старт

1. **Установи cron jobs:**
   ```bash
   python3 scripts/setup.py install
   ```

2. **Начни отвечать на вопросы:**
   - Каждые 30 минут (8:00-22:00) получаешь: "Чем занимался последние 30 минут?"
   - Отвечай голосом или текстом
   - Данные автоматически логируются в Obsidian

3. **Получай недельные сводки:**
   - Автоматом в пятницу 19:00
   - Или вручную: `python3 scripts/weekly.py`

## Что внутри

- `config.json` — настройки времени, интервалов, путей
- `scripts/setup.py` — установка/удаление cron jobs
- `scripts/weekly.py` — генерация недельной статистики
- `references/categories.md` — категории для группировки

## Настройка

Отредактируй `config.json` под себя:
- Рабочие часы (start_hour, end_hour)
- Интервал опросов (interval_minutes)
- Путь до дневника Obsidian
